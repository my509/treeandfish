# 🌿 Trồng Cây Nuôi Cá 🐟

Game mobile đơn giản - Trồng cây, nuôi cá và thu hoạch!

## 📱 Cách tải APK

### Phương pháp 1: Tải trực tiếp
1. Vào [Releases](https://github.com/YOUR_USERNAME/farm-fish-game/releases)
2. Tải file APK mới nhất
3. Cài đặt trên điện thoại Android

### Phương pháp 2: Tạo APK online
1. Truy cập: https://www.pwabuilder.com/
2. Nhập URL GitHub Pages của bạn
3. Chọn "Android" → "Generate"
4. Tải APK về cài đặt

## 🎮 Cách chơi
- Trồng cây (20 vàng) → Đợi phát triển → Thu hoạch
- Nuôi cá (cần thức ăn) → Cho ăn → Thu hoạch
- Mua thức ăn (10 vàng = 3 thức ăn)

## 🚀 Deploy lên GitHub Pages
1. Fork repository này
2. Vào Settings → Pages
3. Chọn branch main → Save
4. Game sẽ có tại: https://YOUR_USERNAME.github.io/farm-fish-game/
